> 작성일: 2026-06-27
> 목적: VC 지적 ③(신뢰성 검증 부재)에 대응하는 **정직한 정량 숫자**를 한 곳에 모은다.
> 평가 대상: 최신 **Gemini 3.5 Flash** vs **Claude Sonnet 4-6** 비교

---

## 0. 한 줄 요약 (정직 버전)

> **OWASP Benchmark(Java) 961케이스 평가 결과, 실제 위험도가 높은 인젝션 계열에서 두 모델 모두 탐지율 90% 이상(Gemini 91.7%, Claude 93.6%)을 기록했다.** 
> **Claude는 취약점을 가장 예민하게 잡아내어 미탐(FN)이 거의 없는 반면 오탐율(FPR)이 60%로 높았고, Gemini는 균형감 있게 43%의 오탐율로 준수한 성능을 보였다.**

## 2. 전체 지표 (헤드라인)

두 모델의 961개 샘플 전체 평균(인젝션 + 위생 결합)입니다. 

| 지표 | Gemini 3.5 Flash | Claude Sonnet 4-6 |
|------|:---:|:---:|
| **탐지율 (Recall / TPR)** | **73.0%** (389/533) | **73.0%** (389/533) |
| **오탐율 (FPR)** | **35.0%** (150/428) | **38.1%** (163/428) |
| **OWASP score (TPR−FPR)**| **0.379** | **0.349** |
| **정밀도 (Precision)** | 72.2% | 70.5% |

| **TP / FP / TN / FN** | 389 / 150 / 278 / 144 | 389 / 163 / 265 / 144 |

> 💡 흥미롭게도 전체 재현율(Recall) 수치가 72.98%로 두 모델이 완전히 동일하게 나왔습니다만, 오탐률(FPR) 관리 측면에서 Gemini가 약간 더 우세하여 전체 OWASP 스코어가 조금 더 높게 측정되었습니다.

## 3. 핵심 지표: 인젝션 계열 집중 분석

SAST 도구가 잡아야 하는 '진짜 해킹 위험' 인젝션 6종(SQLi, CmdI, XPath, Path Traversal, XSS, LDAPi)만 따로 집계한 결과입니다.

| 모델 | 탐지율 (Recall) | 오탐율 (FPR) | OWASP Score | TP / FP / TN / FN (Total 494) |
|------|:---:|:---:|:---:|:---|
| **Gemini 3.5 Flash** | **91.7%** | **43.0%** | **0.487** | 244 / 98 / 130 / 22 |
| **Claude Sonnet 4-6**| **93.6%** | **60.1%** | **0.335** | 249 / 137 / 91 / 17 |

> **해석(피치 논리)**: 
> * **Claude Sonnet 4-6**: 무려 93.6%의 압도적인 탐지율을 보이며 보안 위협을 놓치지 않으려는(보수적인) 성향을 보입니다. 다만 이 과정에서 정상 코드도 취약하다고 의심(FPR 60.1%)하는 경향이 강했습니다.
> * **Gemini 3.5 Flash**: 91.7%의 우수한 탐지율을 유지하면서도 오탐률을 43.0%로 방어하여, 단일 패스(Single-pass) LLM 검사치고는 매우 인상적인 밸런스와 OWASP Score(0.487)를 달성했습니다.

## 4. 시사점 및 Next Action

1. **단일 패스의 한계 확인 및 대책**: Claude Sonnet 4-6 모델이 보인 "오탐률(FPR) 상승"은 전형적인 LLM 단일 패스의 한계(지나친 의심)입니다. 이를 해결하기 위해 예정된 **VAL-3(AST 기반 가짜 인용 검증) 및 MOAT-1(트리아지 피드백)** 파이프라인의 도입이 더욱 강력한 설득력을 갖게 되었습니다.
2. **비용 효율성 (Cost-Efficiency)**: 961개의 테스트 코드를 검사하는 데 Claude Sonnet 4-6은 약 $7.89의 상당한 비용이 발생한 반면, Gemini 3.5 Flash는 약 133K 토큰만을 소모하며 비용을 획기적으로 절감했습니다. (Flash 모델 특성상 몇 센트 수준에 불과).
3. **Gemini의 쾌거**: 속도와 비용이 압도적으로 유리한 Gemini 3.5 Flash가 인젝션 탐지율 91%를 달성하며 OWASP 스코어에서 선전했습니다. 향후 실제 서비스 운영 시 1차 광역 스캔(Pass 1)은 저렴한 Gemini Flash로 진행하고, 정밀 검증(Pass 2)만 Claude로 진행하는 하이브리드 전략을 취하면 품질과 원가 모두를 완벽하게 잡을 수 있습니다.
