- `깨비_SparkClaw_지원서_v2.md`, `깨비_마스터_전략정리.md`, `깨비_핸드오프_브리프_v2.md` 전부 신버전으로 교체. 오탐율(43%/60%)·전체 Recall 73% 병기 유지. 옛 수치 잔존 0건 확인.
